# Hollowreach

A 3D first-person exploration game built with [Bevy](https://bevyengine.org/) and Rust.

## Features

- First-person WASD movement and mouse look
- Interactable objects and NPCs (press E)
- Atmospheric 3D environment with lighting and shadows
- NPC dialogue system (console output)

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look around |
| E | Interact |
| Esc | Toggle cursor lock |

## Building & Running

```bash
cargo run
```

## Requirements

- Rust 1.80+
- Windows 10/11 (primary target)
